# 変更内容の確認 (Walkthrough)

## 実装された機能
1. **設定管理 (`src/config.js`)**: 検索クエリや宛先を一箇所で管理。
2. **ヘッドライン抽出 (`src/parser.js`)**: メール本文から有効な行を抽出し、クリーニングする機能。
3. **メインワークフロー (`src/main.js`)**: Gmailの検索、解析、レポート作成、送信、メール削除の一連の流れ。

## 使用方法
1. 作成された `.js` ファイルの内容を Google Apps Script プロジェクト（`.gs` ファイル）にコピー&ペーストしてください。
2. `src/config.js` 内の `SEARCH_QUERY` と `RECIPIENT_EMAIL` をご自身の環境に合わせて変更してください。
3. `processGovernmentNewsEmails` 関数を実行して動作を確認してください。
    - 初回実行時は権限の承認が必要です。

## 検証手順
1. テスト用のメールを自分宛てに送信する（件名や本文をターゲットに合わせて作成）。
2. スクリプトを実行する。
3. 指定した宛先に「ヘッドライン＋検索リンク」のメールが届いているか確認する。
4. 元のメールがゴミ箱に移動しているか確認する。
